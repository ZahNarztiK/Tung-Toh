Default Response JSON
{
	"message_code" : "-1",
	"message_description" : "",
	"data" : {}
}

All data will be received & returned in JSON named "data", except register, login, & GET input.
Data: Method is required in GET form; method=[add/edit/remove]

* Required
##### Member #####

member.php?method=register, member.php?method=login [POST]
* "email" (string)
* "password" (string)

member.php?method=verifySession [POST]
* "session_id" (string)


Response (Succeeded)
"data" : {
	"verified" : true (bool)
	"session_id" (string)
	"info" (JSON from getProfile.php)
}

Response (Failed)
"data" : {
	"verified" => false (bool)
}

----------

member.php?method=logout

Response
"data" : {
	"logged_out" (bool: logged in before)
}

	"firstname" (string)

	"level" (int)
	"points" (int)
	"profile_image" (string: url)
	"email" (string)
}
