Default Response JSON
{
	"message_code" : "-1",
	"message_description" : "",
	"data" : {}
}

All data will be received & returned in JSON named "data", except register, login, & GET input.

* Required
##### Member #####

register.php, login.php [POST]
* "email" (string)
* "password" (string)

verifySession.php [POST]
* "session_id" (string)


Response (Succeeded)
"data" : {
	"verified" : true (bool)
	"session_id" (string)
	"info" (JSON from getProfile.php)
}

Response (Failed)
"data" : {
	"verified" => false (bool)
}

----------

logout.php

Response
"data" : {
	"logged_out" (bool: logged in before)
}

#####Data##### [Log in required; Admin required for adding data]

getProfile.php [GET]
- member_id (int: default - current user)

Response
"data" : {
	"verified" (int) [0: Not verified, 1: Verified member, 2: Admin]
	"firstname" (string)
	"lastname" (string)
	"level" (int)
	"points" (int)
	"profile_image" (string: url)
	"email" (string)
}
